THREDDS Server - Status

Comments to THREDDS mail list

The THREDDS Server distribution is a WAR file which contains a number of servlets that provide various THREDDS services. For instance, the THREDDS server contains a THREDDS catalog validation service, a THREDDS catalog generation service, and a THREDDS catalog to HTML converter.

Current Stable Release

THREDDS Server version 0.2c (released 5 August 2004).

The THREDDS server WAR file version 0.2c is available for download, thredds.war. For information on installing and running a THREDDS Server, go to the installation page.
