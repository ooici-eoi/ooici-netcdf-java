ooi-netcdf-java - OOI Version of the NetCDF-Java Library
==================================================

WARNING - this is a large repository and will take some time to download

Get ooi-netcdf-java with
::
	git clone git@amoeba.ucsd.edu:ooi-netcdf-java.git
	cd ooi-netcdf-java


Dependencies
============

ooi-netcdf-java requires the following projects:
	ioncore-java - git@amoeba.ucsd.edu:ioncore-java.git
	
****The dependencies should be obtained and compiled prior to continuing****

Compile the library and create the jar file:
::
	ant makeFullJar
